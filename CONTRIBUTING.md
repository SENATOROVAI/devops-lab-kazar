## Как участвовать в проекте

1. Создайте issue, если хотите предложить улучшение или сообщить об ошибке.
2. Создайте новую ветку от `develop`:  
   `git checkout -b feature/название-ветки`
3. Внесите изменения.
4. Сделайте коммит с понятным сообщением:  
   `git commit -m "Add: описание изменения"`
5. Запушьте ветку:  
   `git push origin feature/название-ветки`
6. Откройте Pull Request в ветку `develop` или `main`.

## Правила оформления коммитов
- Начинайте сообщение с глагола в повелительном наклонении: Add, Fix, Update, Refactor, Docs...
- Делайте небольшие, логически законченные коммиты
- Используйте английский язык (если нет каких-либо иных договорённостей)

Спасибо за ваш вклад в проект!
